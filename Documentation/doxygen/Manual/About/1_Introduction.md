# Introduction

## About
The Widrolo Engine is a proprietary 3D game engine designed to work on Windows and Linux. Its meant to be intuitive to work on for people that just want to code instead of hanging around editors all day long, but also allows designers to edit the world via WEDGE.

## Installation
1. Download the repo: https://github.com/widrolo/Widrolo-Engine3D.git
2. Download CLion: https://www.jetbrains.com/clion/download
3. Install GCC if compiling for Linux, Visual Studio if you're on Windows.
4. Install dependencies like Vulkan and VMA.
5. Open the repo in CLion and run to see what you're missing. Install whatever you're missing.

Note that while CLion is not a strict requirement, the engine is always in the newest version of C++, which Visual Studio might not support.
